Keeping state from 2 sources

In this example, there are 2 competing states. Both states must agree to trigger a state change AND the change must be incremental. So in the initial case, at state 1, the next state is state 2 and this can only occur when both internal and external states are 2. The exception to this rule is that you can immediately go to state 5 by setting the internalState to 5. State 5 is the final state.
