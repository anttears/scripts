Growler Test Page

This code is designed to generate system alerts that look a bit like Growl on the Mac. A small box (which you can style on a per growl basis) appears in the top left of the site, containing your custom message

Growls can be added to the html of the page and shown at page load, or generated by javascript.

Seems to work in mobern browsers and IE down to 6

Usage

Library free js

Simply add the js file to your site and change the value of cssHook to be the id of the element you want the growlList to be added to. Its usually the first main element on the page and needs to be position relative. If you want growls to show on page load, add the growl markup. The js checks for this at the start and will remove the growls after the designated time period.

yourNameSpace.growl(className, "Some test for the Header", "Content Text");
